# Weeslack Lightshot
Run /slack upload with last lightshot file

### Requirements
WeeChat
WeeSlack

## Instalation
```
wget https://raw.githubusercontent.com/dzfweb/weeslack-lightshot/master/lightshot.py
cp lightshot.py ~/.weechat/python/autoload
```

Define the lightshot path
```
/set plugins.var.python.lightshot.path "/mnt/c/Users/dougl/Documents/Lightshot"
```

## Usage
```
/lightshot
```
